## 4. تثبيت المشروع محلياً

لإعداد المشروع على جهازك، اتبع هذه الخطوات:

### المتطلبات الأساسية
- Node.js (الإصدار 18 أو أحدث)
- npm أو yarn أو pnpm

### الخطوات
1. **نسخ المستودع:**
   ```bash
   git clone <your-repository-url>
   cd my-portfolio
   ```

2. **تثبيت التبعيات:**
   ```bash
   npm install
   ```

3. **تشغيل خادم التطوير:**
   ```bash
   npm run dev
   ```

4. **فتح المتصفح:**
   اذهب إلى الرابط الذي يظهر في الـ Terminal (غالباً `http://localhost:5173`)

### أوامر إضافية
- `npm run build`: بناء نسخة إنتاج (Production) من الموقع في مجلد `dist/`
- `npm run preview`: معاينة نسخة الإنتاج محلياً قبل النشر

## 5. هياكل المجلدات

```text
my-portfolio/
├── public/                 # الملفات الثابتة (الصور، الخ)
│   ├── images/
│   │   ├── profile-headshot.jpg
│   │   └── ...
│   ├── robots.txt
│   └── sitemap.xml
├── src/
│   ├── components/
│   │   ├── sections/      # أقسام الموقع الرئيسية (Hero, About, Projects, إلخ)
│   │   ├── ui/            # المكونات الصغيرة والمُستخدمة مرة أخرى (CookieConsent)
│   │   └── Header.tsx, Footer.tsx
│   ├── contexts/          # إدارة الحالة (اللغة، الوضع الليلي)
│   ├── hooks/             # Hooks مخصصة
│   ├── i18n.ts            # ملف الترجمة (العربية والإنجليزية)
│   ├── App.tsx            # المكون الرئيسي
│   └── main.tsx           # نقطة الدخول
├── index.html             # ملف HTML الأساسي (SEO, Meta Tags)
├── vite.config.ts
└── package.json
```

## 6. تخصيص المحتوى

يمكنك تعديل محتوى الموقع بسهولة من خلال ملف `src/i18n.ts`:
- جميع النصوص موجودة في هذا الملف، مرتبة حسب اللغة (العربية والإنجليزية)
- يمكنك تغيير الاسم، المعلومات الشخصية، المشاريع، المهارات، آراء العملاء، وغيرها

### تحديث الصور
قم باستبدال الصور في مجلد `public/images/` بصورك الخاصة، مع الحفاظ على نفس الأسماء أو تحديثها في الكود.
